# Comic-Story-Builder
An AI Agent generated app using Replit.

## Description
A digital playground that allows users to write a short story and automatically converts each sentence into a visually consistent comic book panel. 

## Overview
An intuitive Comic creation app that turns your imagination into visual stories. Its purpose is to to provide a creative outlet. Writing and visualizing stories can be therapeutic, helping users process emotions or simply escape into a world of their own making. I built this app as a beginner project to help learn computer programming and also to experiment with art.

## Features
Main Functionalities:
- Character Locking
- Drag-and-Drop Lettering
- Panel Layout Templates
- Style Filters
- Export to PDF
- Responsive layout (works on mobile and desktop)

## Tech Stack
- **Language:** Typescript
- **Frontend:** React, Vite, Tailwind CSS, Radix UI, React Hook Form
- **Backend:** Node.js, Express
- **Database:** Drizzle ORM
- **Tools:** drizzle-kit, npm, Git, Replit
